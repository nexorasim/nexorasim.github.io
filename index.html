SM-DP+ Provisioning Flow

End-to-end GSMA compliant remote provisioning process for consumer and IoT devices

1

Discovery

Device retrieves SM-DP+ address from GSMA root DNS or local discovery service

2

Authentication

Mutual TLS authentication using EUM and SM-DP+ certificates per SGP.22

3

Profile Download

Secure profile package download with GSMA-defined encryption and integrity

4

Installation

LPA installs and activates profile with confirmation to SM-DP+

5

Management

OTA profile management including enable/disable/delete operations
